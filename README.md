# RESOLVA Assist – AI Claims Copilot

Transforming 23 years of frontline claims expertise into AI guidance for the next generation of insurance professionals.

## Overview

RESOLVA Assist is an AI-powered decision support system designed for junior automobile insurance claims handlers.

It helps users:

- Identify missing information
- Structure the next step
- Apply consistent claims handling practices
- Learn from real-world expertise

Built with Dify and AI-assisted development.

## Why I built this

After spending 23 years handling automobile insurance claims, I wanted to preserve practical frontline expertise and make it accessible to the next generation through AI.
